# 🛒 Superstore Sales Analysis

A data cleaning and visualization project using Python and Google Looker Studio.

## 📊 Live Dashboard
👉 [Click here to view the dashboard](https://datastudio.google.com/reporting/2faced1c-a9e2-473f-8d27-e8ce5aeebe2f)

## 🧰 Tools Used
- Python (pandas, numpy) — data cleaning
- Google Colab — coding environment
- Google Looker Studio — dashboard & visualization
- Kaggle — data source

## 📁 Dataset
- **Source:** Superstore Sales Dataset (Kaggle)
- **Rows:** 9,994 orders
- **Period:** 2014 to 2017

## 🧹 What I Cleaned
- Removed duplicate rows
- Fixed missing values
- Corrected data types (dates, numbers)
- Standardized column names
- Extracted year, month, quarter from order date
- Renamed inconsistent column (sub-category)

## 📈 Dashboard Insights
- 💰 Total Sales: $2,297,200
- 📦 Total Orders: 5,009
- 📈 Total Profit: $286,397
- 🏆 Top Sub-Category: Phones & Chairs
- 🌍 Top Region: West (31.6%)
- 💡 Most Profitable Category: Technology
